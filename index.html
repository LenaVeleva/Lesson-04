Команды, которые проделываю в терминале:

Делаю первые команды на терминале

  1. git --version - команда, которой проверяю установлен ли GIT и его версия
  2. git init - инициализация репозитория, один раз для проекта

Сохранения состояния нужных файлов и папки

  1. git add - сначала делаем эту команду для указания каких именно файлов/папок, мы сохранить в истории
